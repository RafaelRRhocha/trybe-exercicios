5 motivos para você aprender JavaScript!
------------------------------------------

O JavaScript é uma linguagem de programação muito poderosa utilizada em todo o mundo!

O uso mais convencional do JavaScript é no contexto do nosso navegador, juntamente com HTML e CSS.

Por que o JavaScript é tão valorizado?
----------------------------------------

O JavaScript te permite dar dinamismos às páginas. É ele quem garante que sua página tenha a habilidade de reagir às ações das pessoas usuárias.

5 motivos para você aprender JavaScript
-----------------------------------------

#1 Linguagem muito popular e adotada pelas principais empresas de tecnologia do mundo.

#2 Constante evolução, principalmente por ter uma comunidade engajada.

#3 muitas opções de bibliotecas para se usar.

#4 Pode ser muito bem utilizado tanto no front quanto no back end.

#5 Sintaxe mais educacional e simples do que as outras.

#6 É fácil de aprender.

#7 Possibilita utilizar com programação orientada a objetos.

#8 Contribui com a absorção de outras linguagens.

#9 Confere abundância de frameworks.

#10 Uma linguagem de fácil iniciação para os que se iniciam na programação.

#11 Permite fácil manipulação com objetos terceiros.

#12 Uma boa linguagem base para manipular com outras linguagens.

#13 Biblioteca ampla de frameworks.

#14 linguagem didática.

#15 da a opcao trabalhar com programação orientada a objetos.

#16 E uma boa linguagem de introdução.

#17 Apresenta muitas faces de trabalho.
